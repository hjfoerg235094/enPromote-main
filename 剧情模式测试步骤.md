```bash

cd server

# 启动前端服务（新终端）
cd frontend
npm run dev
```

### 第二步：访问剧情模式

1. 打开浏览器访问 `http://localhost:5173`
2. 登录系统（如果未登录）
3. 点击导航栏中的"剧情模式"
4. 应该看到"特工行动：代号暗影"剧情卡片

3. 点击"第一章：潜入"开始章节
4. 查看章节背景和任务列表

## 🔧 常见问题

### Q1: 点击"剧情模式"后页面空白
**A**: 检查浏览器控制台是否有错误，确认 StoryMode_new.vue 组件是否正确加载

### Q2: 无法看到剧情列表
**A**: 检查后端服务是否启动，确认 `/api/story/list` 接口是否正常

### Q3: 点击章节后无法进入
**A**: 检查路由参数是否正确，确认 ChapterView_new.vue 组件是否正确加载

## 📚 API接口说明

### 1. 获取剧情列表
```
GET /api/story/list
```

### 2. 获取剧情详情
```
GET /api/story/:storyId
```

### 3. 获取章节详情
```
GET /api/story/:storyId/chapter/:chapterId
```

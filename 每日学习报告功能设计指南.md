# 每日学习报告功能设计指南

## 1. 功能概述

### 1.1 功能简介
每日学习报告功能模块旨在为用户提供每日学习成果的可视化展示，帮助用户了解自己的学习进度、成就和待改进之处，从而提升学习动力和效果。

### 1.2 核心目标 
- 直观展示用户每日学习数据
- 提供学习成就和进步的可视化反馈
- 激励用户持续学习
- 帮助用户识别学习中的薄弱环节

## 2. 功能需求

### 2.1 基础数据统计
- **学习时长统计**
  - 总学习时长
  - 各模块学习时长（词汇、听力、拼写、AI练习等）

- **学习内容统计**
  - 新学单词数量
  - 复习单词数量
  - 拼写练习正确率
  - 听力练习完成度
  - AI对话练习次数

- **学习成就统计**
  - 连续学习天数
  - 累计学习天数
  - 获得徽章/成就数量

### 2.2 数据可视化
- **学习时长分布图**
  - 使用环形图或柱状图展示各模块学习时长占比

- **学习进度趋势图**
  - 使用折线图展示最近7天/30天学习时长变化

- **单词掌握度分析**
  - 使用雷达图展示词汇、听力、拼写、阅读、写作等能力维度

- **学习日历**
  - 使用日历热力图展示每月学习打卡情况

### 2.3 个性化反馈
- **学习建议**
  - 根据学习数据生成个性化学习建议
  - 推荐需要加强的练习模块

- **成就解锁**
  - 达成特定学习目标时解锁新成就
  - 显示成就解锁动画和奖励

### 3.2 组件设计

#### 3.2.1 学习总览卡片
- **布局**: 4个数据卡片横向排列
- **样式**:
  - 卡片背景: 白色
  - 圆角: 12px
  - 阴影: 0 2px 8px rgba(0, 0, 0, 0.1)
  - 悬停效果: 上移2px，阴影增强

- **内容**:
  - 学习时长: 显示总学习时长（如"45分钟"）
  - 学习单词: 显示新学+复习单词总数（如"30个"）
  - 正确率: 显示今日练习平均正确率（如"85%"）
  - 连续天数: 显示连续学习天数（如"7天"）

#### 3.2.2 学习时长分布图
- **图表类型**: 环形图
- **数据维度**: 词汇、听力、拼写、AI练习等
- **交互**: 
  - 悬停显示具体时长和占比
  - 点击图例可隐藏/显示对应数据

#### 3.2.3 学习进度趋势图
- **图表类型**: 折线图
- **时间维度**: 最近7天/30天（可切换）
- **数据维度**: 学习时长、学习单词数
- **交互**:
  - 悬停显示具体日期的数据
  - 点击图例可切换显示不同数据维度

#### 3.2.4 学习日历
- **显示方式**: 日历热力图
- **颜色编码**:
  - 深紫色: 学习时长>60分钟
  - 中紫色: 学习时长30-60分钟
  - 浅紫色: 学习时长<30分钟
  - 灰色: 未学习
- **交互**:
  - 点击日期显示当日详细学习数据

#### 3.2.5 学习建议卡片
- **布局**: 列表形式
- **内容**:
  - 根据学习数据生成个性化建议
  - 每条建议包含图标、标题和描述
  - 提供"开始练习"按钮

#### 3.2.6 成就展示卡片
- **布局**: 网格形式
- **内容**:
  - 已解锁成就: 彩色图标+名称+解锁日期
  - 未解锁成就: 灰色图标+名称+解锁条件
- **交互**:
  - 点击成就查看详情
  - 新解锁成就有动画效果

## 4. 数据结构设计

### 4.1 每日学习报告数据结构
```typescript
interface DailyStudyReport {
  date: string;                    // 日期，格式: YYYY-MM-DD
  totalStudyTime: number;           // 总学习时长（分钟）
  moduleStudyTime: {               // 各模块学习时长
    vocabulary: number;            // 词汇学习时长
    listening: number;             // 听力练习时长
    spelling: number;              // 拼写练习时长
    aiPractice: number;            // AI练习时长
  };
  wordsLearned: {                  // 单词学习统计
    newWords: number;              // 新学单词数
    reviewWords: number;          // 复习单词数
  };
  practiceStats: {                 // 练习统计
    spellingAccuracy: number;      // 拼写正确率（0-100）
    listeningCompletion: number;   // 听力完成度（0-100）
    aiPracticeCount: number;       // AI练习次数
  };
  achievements: {                  // 成就统计
    continuousDays: number;        // 连续学习天数
    totalDays: number;             // 累计学习天数
    unlockedAchievements: string[] // 已解锁成就ID列表
  };
}
```

### 4.2 学习进度趋势数据结构
```typescript
interface StudyProgressTrend {
  date: string;                    // 日期，格式: YYYY-MM-DD
  studyTime: number;               // 学习时长（分钟）
  wordsCount: number;              // 学习单词数
}
```

### 4.3 成就数据结构
```typescript
interface Achievement {
  id: string;                      // 成就ID
  name: string;                    // 成就名称
  description: string;             // 成就描述
  icon: string;                    // 成就图标
  unlocked: boolean;               // 是否已解锁
  unlockedDate?: string;           // 解锁日期，格式: YYYY-MM-DD
  progress?: {                     // 解锁进度（未解锁时显示）
    current: number;               // 当前进度
    target: number;                // 目标值
  };
}
```

## 5. API接口设计

### 5.1 获取每日学习报告
```
GET /api/report/daily
参数:
  - date: string (可选，格式: YYYY-MM-DD，默认为今天)

响应:
{
  "code": 200,
  "message": "success",
  "data": DailyStudyReport
}
```

### 5.2 获取学习进度趋势
```
GET /api/report/trend
参数:
  - startDate: string (格式: YYYY-MM-DD)
  - endDate: string (格式: YYYY-MM-DD)

响应:
{
  "code": 200,
  "message": "success",
  "data": StudyProgressTrend[]
}
```

### 5.3 获取成就列表
```
GET /api/report/achievements

响应:
{
  "code": 200,
  "message": "success",
  "data": Achievement[]
}
```

### 5.4 获取学习建议
```
GET /api/report/suggestions

响应:
{
  "code": 200,
  "message": "success",
  "data": {
    "suggestions": [
      {
        "id": string,
        "type": string,           // 建议类型: vocabulary, listening, spelling, aiPractice
        "title": string,
        "description": string,
        "actionUrl": string       // 操作链接
      }
    ]
  }
}
```

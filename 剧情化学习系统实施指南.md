## 🚀 实施步骤

### 第一步：后端集成（必须）

#### 1.1 在主应用中注册剧情路由

编辑 `server/app.js`，添加剧情路由：

```javascript
const storyRouter = require('./router/story');
app.use('/api/story', storyRouter);
```

#### 1.2 确保数据库模型已创建

StoryProgress.js 中已经定义了 MongoDB 模型，确保你的 MongoDB 连接正常。

### 第二步：前端路由配置（必须）

#### 2.1 添加剧情相关路由

编辑 `frontend/src/router/index.ts`，添加：

```typescript
{
  path: '/story',
  name: 'StoryMode',
  component: () => import('@/views/StoryMode.vue')
},
{
  path: '/story/:storyId',
  name: 'StoryDetail',
  component: () => import('@/views/StoryMode.vue')
},
{
  path: '/chapter/:storyId/:chapterId',
  name: 'ChapterView',
  component: () => import('@/views/ChapterView.vue')
}
```

#### 2.2 在导航菜单中添加入口

在主界面添加"剧情模式"入口，链接到 `/story` 路由。

### 第三步：AI集成（推荐）

#### 3.1 集成对话任务的AI功能

在 `DialogueTask.vue` 中的 `getAIResponse` 方法中，集成实际的AI API调用：

```typescript
async function getAIResponse(userMessage: string) {
  try {
    const response = await fetch('/api/aiApi/taskChat', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        message: userMessage,
        systemPrompt: props.task.systemPrompt,
        scene: props.task.scene
      })
    });

    const data = await response.json();
    return data.content;
  } catch (error) {
    console.error('AI响应失败:', error);
    throw error;
  }
}
```

#### 3.2 生成听力材料

使用AI生成听力材料和问题，可以创建一个后端接口来生成这些内容。

## 🔧 常见问题

### Q1: 剧情列表加载失败
A: 检查 `storyScenes.json` 文件路径是否正确，确保文件格式有效。

### Q2: 任务无法完成
A: 检查任务完成标准是否合理，验证任务数据是否正确提交。

### Q3: 进度未保存
A: 检查数据库连接，确保用户ID正确传递。

### Q4: AI响应失败
A: 检查API配置，确保API Key有效，网络连接正常。


# 词汇学习功能使用指南

本指南详细说明了如何使用新实现的词汇学习功能，实现第一关词汇练习的持续学习。

## 功能概述

新实现的词汇学习功能包含以下核心功能：

1. **学习进度跟踪** - 自动跟踪用户的学习进度和统计
2. **智能单词推荐** - 根据学习进度推荐合适的单词
3. **间隔重复复习** - 基于艾宾浩斯遗忘曲线安排复习
4. **章节管理** - 按章节组织词汇，支持自动章节切换
5. **学习统计** - 提供详细的学习数据和进度可视化

## 系统架构

### 后端组件

1. **vocabularyLearningService** - 词汇学习服务
   - 管理学习进度
   - 处理单词学习记录
   - 计算复习时间
   - 管理章节切换

2. **vocabularyLearning路由** - API接口
   - 获取学习进度
   - 获取学习单词列表
   - 更新学习进度
   - 记录单词学习结果
   - 获取复习单词
   - 切换章节
   - 获取学习统计

### 前端组件

1. **vocabularyLearning.vue** - 词汇学习页面
   - 学习模式选择
   - 单词学习界面
   - 学习进度显示
   - 复习模式

2. **vocabularyLearning API** - API接口封装
   - 提供所有后端API的调用方法

## 使用流程

### 1. 初始化系统

#### 步骤1：导入词汇数据

使用词汇库管理工具导入词汇数据到数据库：

```bash
cd server/scripts
npm install
npm run import -- --file ../vocabulary/CET-4.json --level CET-4
```

#### 步骤2：优化数据库

运行数据库优化工具，确保索引正确创建：

```bash
npm run optimize
```

#### 步骤3：启动服务器

```bash
cd server
node app.js
```

### 2. 用户学习流程

#### 开始学习

1. 用户登录系统
2. 进入词汇学习页面（vocabularyLearning.vue）
3. 查看当前学习进度和统计
4. 点击"开始学习"按钮

#### 学习单词

1. 系统显示当前章节的单词列表（默认20个）
2. 用户对每个单词进行判断：
   - **认识** - 标记为已掌握，延长复习间隔
   - **模糊** - 标记为需要复习，1天后复习
   - **不认识** - 标记为需要重点复习，1天后复习

3. 学习完成后，系统自动更新学习进度

#### 复习单词

1. 点击"开始复习"按钮
2. 系统根据间隔重复算法提供需要复习的单词
3. 用户对每个单词进行判断
4. 系统更新复习时间和间隔

#### 章节管理

1. 查看当前章节进度
2. 完成当前章节后，系统自动切换到下一章节
3. 用户也可以手动切换到下一章节

### 3. 学习进度管理

#### 进度存储

学习进度存储在User模型的cet4.position字段中，格式为"章节:索引"，例如：
- "A:0" - A章节第0个单词
- "A:20" - A章节第20个单词
- "B:0" - B章节第0个单词

#### 进度更新

每次学习完成后，系统会：
1. 更新当前位置
2. 记录学习单词数
3. 更新连续学习天数
4. 添加学习历史记录

#### 复习算法

系统使用间隔重复算法安排复习：

- **认识** - 复习间隔逐渐增大：1天 → 3天 → 7天 → 14天 → 30天
- **模糊** - 1天后复习
- **不认识** - 1天后复习

## API接口说明

### 1. 获取学习进度

```
GET /api/vocabulary-learning/progress
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "currentChapter": "A",
    "currentIndex": 0,
    "chapterTotal": 100,
    "chapterProgress": 0,
    "totalLearned": 0,
    "todayLearned": 0,
    "streakDays": 0,
    "lastStudyDate": "2024-01-01T00:00:00.000Z"
  }
}
```

### 2. 获取学习单词列表

```
GET /api/vocabulary-learning/words?chapter=A&startIndex=0&count=20
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "words": [...],
    "total": 100,
    "currentIndex": 0,
    "hasMore": true,
    "chapter": "A"
  }
}
```

### 3. 更新学习进度

```
POST /api/vocabulary-learning/progress
```

**请求体**：
```json
{
  "studiedWords": 20,
  "chapter": "A",
  "newIndex": 20
}
```

### 4. 记录单词学习结果

```
POST /api/vocabulary-learning/word
```

**请求体**：
```json
{
  "word": "abandon",
  "status": "know"
}
```

### 5. 获取复习单词

```
GET /api/vocabulary-learning/review?limit=20
```

### 6. 切换章节

```
POST /api/vocabulary-learning/next-chapter
```

### 7. 获取学习统计

```
GET /api/vocabulary-learning/statistics
```

## 配置选项

### 学习参数

可以在vocabularyLearningService.js中调整以下参数：

- **每次学习单词数** - 默认20个
- **复习间隔算法** - 根据学习状态调整复习间隔
- **章节列表** - 支持的章节（A-Z）

### 前端配置

可以在vocabularyLearning.vue中调整以下参数：

- **学习模式** - 练习或复习
- **每批单词数** - 默认20个
- **进度显示** - 进度条和计数器

## 常见问题

### 1. 如何重置学习进度？

可以通过以下方式重置学习进度：

```javascript
// 在MongoDB中执行
db.users.updateOne(
  { _id: userId },
  { $set: { 'cet4.position': 'A:0' } }
)
```

### 2. 如何调整学习单词数？

修改vocabularyLearning.vue中的loadWordList函数：

```javascript
const res = await getLearningWords({
  chapter,
  startIndex,
  count: 30 // 修改为30个
});
```

### 3. 如何自定义复习间隔？

修改vocabularyLearningService.js中的calculateNextReviewTime方法：

```javascript
calculateNextReviewTime(status, reviewCounts) {
  // 自定义复习间隔算法
  ...
}
```

### 4. 如何添加新的词汇等级？

1. 准备新的词汇JSON文件
2. 使用词汇库管理工具导入：
```bash
npm run import -- --file ../vocabulary/CET-6.json --level CET-6
```

## 性能优化

### 1. 数据库索引

确保以下索引已创建：

- UserWord: { userId: 1, word: 1 }
- UserWord: { userId: 1, nextReviewTime: 1 }
- Word: { word: 1 }
- Word: { level: 1, chapter: 1 }

### 2. 缓存策略

使用vocabularyService的缓存功能：

```javascript
// 缓存24小时
vocabularyService.loadVocabulary('CET-4');

// 手动清除缓存
vocabularyService.clearCache('CET-4');
```

### 3. 批量操作

使用批量API减少请求次数：

```javascript
// 批量记录单词学习结果
batchRecordWordLearning([
  { word: 'abandon', status: 'know' },
  { word: 'aboard', status: 'vague' },
  ...
]);
```

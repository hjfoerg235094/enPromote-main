# 社区排名和好友系统设计文档

## 一、项目概述

### 1.1 目标
为enPromote英语学习平台添加社区排名和好友系统，通过社交互动和良性竞争提升用户学习动力和平台粘性。

### 1.2 核心价值
- **提升用户粘性**：通过社交关系和排名竞争增加用户使用频率
- **增强学习动力**：好友互动和排名激励促进持续学习
- **构建学习社区**：形成互助学习的良好氛围
- **促进用户增长**：通过社交网络效应吸引新用户

### 1.3 设计原则
- **隐私优先**：用户可控制自己的数据可见性
- **公平竞争**：建立防作弊机制确保排名公平
- **渐进增强**：分阶段实施，优先实现核心功能
- **性能优先**：使用缓存优化排行榜查询性能

### 3.3 缓存设计

#### 3.3.1 Redis缓存策略

**排行榜缓存**
```
Key: ranking:{type}:{period}:{date}
Type: Sorted Set
Value: userId:score
TTL: 1小时
```

**好友列表缓存**
```
Key: friends:{userId}
Type: List
Value: friendId
TTL: 30分钟
```

**用户排名缓存**
```
Key: user:rank:{userId}:{type}:{period}:{date}
Type: String
Value: rank
TTL: 1小时
```

**在线状态缓存**
```
Key: user:online:{userId}
Type: String
Value: timestamp
TTL: 5分钟
```

#### 3.3.2 缓存更新策略

- **排行榜缓存**
  - 定时任务每小时更新一次
  - 用户学习数据变化时异步更新
  - 使用Redis Sorted Set维护排名

- **好友列表缓存**
  - 添加/删除好友时更新
  - 定期刷新（30分钟）
  - 使用LRU策略管理缓存

- **在线状态缓存**
  - 用户活动时更新
  - 心跳检测（5分钟）
  - 过期自动删除

### 4.3 状态管理

#### 4.3.1 friendStore.ts
```typescript
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'
import * as friendApi from '@/api/friend'

export const useFriendStore = defineStore('friend', () => {
  const friends = ref([])
  const requests = ref([])
  const loading = ref(false)

  // 获取好友列表
  const fetchFriends = async () => {
    loading.value = true
    try {
      const res = await friendApi.getFriends()
      friends.value = res.data.data
    } finally {
      loading.value = false
    }
  }

  // 获取好友请求
  const fetchRequests = async () => {
    const res = await friendApi.getFriendRequests()
    requests.value = res.data.data
  }

  // 发送好友请求
  const sendRequest = async (toUserId: string, message: string) => {
    await friendApi.sendRequest({ toUserId, message })
  }

  // 接受好友请求
  const acceptRequest = async (requestId: string) => {
    await friendApi.acceptRequest({ requestId })
    await fetchFriends()
    await fetchRequests()
  }

  // 拒绝好友请求
  const rejectRequest = async (requestId: string) => {
    await friendApi.rejectRequest({ requestId })
    await fetchRequests()
  }

  return {
    friends,
    requests,
    loading,
    fetchFriends,
    fetchRequests,
    sendRequest,
    acceptRequest,
    rejectRequest
  }
})
```

#### 4.3.2 rankingStore.ts
```typescript
import { defineStore } from 'pinia'
import { ref } from 'vue'
import * as rankingApi from '@/api/ranking'

export const useRankingStore = defineStore('ranking', () => {
  const rankingList = ref([])
  const myRank = ref(null)
  const loading = ref(false)

  // 获取排行榜
  const fetchRanking = async (type: string, period: string) => {
    loading.value = true
    try {
      const res = await rankingApi.getRanking({ type, period })
      rankingList.value = res.data.data.list
      myRank.value = res.data.data.myRank
    } finally {
      loading.value = false
    }
  }

  return {
    rankingList,
    myRank,
    loading,
    fetchRanking
  }
})
```

# 好友系统使用说明

## 一、功能概述

好友系统为enPromote英语学习平台提供了完整的社交功能，包括：
- 添加好友
- 管理好友列表
- 处理好友请求
- 设置好友备注
- 隐私设置管理

## 二、后端实现

### 2.1 数据库模型

系统创建了三个主要的数据模型：

1. **Friendship** (好友关系表)
   - 存储用户之间的好友关系
   - 支持不同的关系状态（pending/accepted/blocked）
   - 记录备注信息

2. **FriendRequest** (好友请求表)
   - 存储好友请求记录
   - 跟踪请求状态（pending/accepted/rejected）
   - 记录请求消息

3. **UserSettings** (用户设置表)
   - 管理用户隐私设置
   - 管理通知偏好
   - 支持个性化配置

### 2.2 API接口

#### 好友管理接口

1. **搜索用户**
   ```
   GET /api/friends/search?username={username}
   ```

2. **发送好友请求**
   ```
   POST /api/friends/request
   Body: { toUserId, message }
   ```

3. **接受好友请求**
   ```
   POST /api/friends/accept
   Body: { requestId }
   ```

4. **拒绝好友请求**
   ```
   POST /api/friends/reject
   Body: { requestId }
   ```

5. **获取好友列表**
   ```
   GET /api/friends/list?page=1&limit=20&sort=-updateTime
   ```

6. **获取好友请求列表**
   ```
   GET /api/friends/requests
   ```

7. **删除好友**
   ```
   DELETE /api/friends/:friendId
   ```

8. **屏蔽好友**
   ```
   POST /api/friends/:friendId/block
   ```

9. **设置好友备注**
   ```
   POST /api/friends/remark
   Body: { friendId, remark }
   ```

#### 用户设置接口

1. **获取用户设置**
   ```
   GET /api/settings
   ```

2. **更新用户设置**
   ```
   PUT /api/settings
   Body: { privacy, notification }
   ```

### 2.3 路由集成

好友和设置路由已经集成到主应用中（server/app.js）：
```javascript
const friendsRouter = require('./router/friends');
const settingsRouter = require('./router/settings');

app.use('/api/friends', friendsRouter);
app.use('/api/settings', settingsRouter);
```

## 四、使用流程

### 4.1 添加好友流程

1. 用户在好友列表页面点击"添加好友"按钮
2. 输入用户名进行搜索
3. 从搜索结果中选择要添加的用户
4. 可选：添加请求消息
5. 发送好友请求
6. 等待对方接受

### 4.2 处理好友请求流程

1. 用户收到好友请求通知
2. 点击查看请求详情
3. 选择"接受"或"拒绝"
4. 系统更新关系状态

### 4.3 管理好友流程

1. 查看好友列表
2. 可以进行以下操作：
   - 查看好友资料
   - 设置好友备注
   - 删除好友
   - 屏蔽好友

### 4.4 隐私设置流程

1. 进入设置页面
2. 调整隐私选项：
   - 是否在排行榜中显示
   - 是否向好友显示学习进度
   - 是否允许好友请求
   - 是否显示在线状态
3. 调整通知选项：
   - 排名变化通知
   - 好友活动通知
   - 好友请求通知
4. 保存设置

## 五、注意事项

1. **权限控制**
   - 所有API都需要用户登录
   - 用户只能操作自己的好友关系
   - 隐私设置会限制某些操作

2. **数据验证**
   - 不能添加自己为好友
   - 不能重复添加已存在的好友
   - 备注长度限制为20个字符

3. **错误处理**
   - 所有操作都有错误处理
   - 友好的错误提示
   - 操作失败时恢复原状态

4. **性能优化**
   - 使用分页加载好友列表
   - 搜索结果限制数量
   - 合理使用缓存

## 七、常见问题

### Q1: 如何添加好友？
A: 在好友列表页面点击"添加好友"按钮，输入用户名搜索后即可发送请求。

### Q2: 如何设置好友备注？
A: 在好友列表中找到该好友，点击编辑图标（✏️），输入备注名称后保存。

### Q3: 如何防止陌生人添加好友？
A: 在设置中关闭"允许好友请求"选项。

### Q4: 删除好友后对方还能看到我的信息吗？
A: 删除好友后，对方将无法查看你的好友专属信息，但仍可能看到公开信息。

### Q5: 好友请求多久过期？
A: 目前好友请求不会自动过期，需要手动处理。
